This project was created a components with react.

## features

- 完整的react组件库开发集成
- 基于官方create-react-app的react开发环境，可以方便引入组件测试
- 集成storybook来管理组件，包括开发、测试、文档、数据。
- 集成组件库编译环境，通过rollup配置编译，npm发布上线


## How to use

```bash

npm i

//组件开发
npm run storybook

//组件打包编译
npm run build:saber

//组件发布
npm publish

//基于app的组件调试
npm start

```
